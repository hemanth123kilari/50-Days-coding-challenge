 Description

This repository contains Python programs that solve common data structure challenges using strings, lists, tuples, and dictionaries. The problems focus on improving logical thinking and problem-solving skills.

Concepts Covered
String manipulation and sliding window technique
List operations and duplicate removal
Tuple operations (max & min)
Dictionary usage and frequency counting
Grouping data using dictionaries
 Problems Included
Longest substring without repeating characters
Remove duplicates from list while preserving order
Find maximum and minimum in a tuple
Character frequency counter using dictionary
Group words by their length
 Key Learnings
Efficient handling of strings and substrings
Maintaining order while removing duplicates
Using built-in functions like max() and min()
Counting elements using dictionaries
Organizing data into groups dynamically
