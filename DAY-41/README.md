## 📊 Financial Performance Analysis – Power BI Project

* This project analyzes company financial data to evaluate revenue, cost, and profitability.
* The dataset includes Region, Product, Revenue, Cost, Profit, Discount, and Customer Type.
* Data transformation and modeling were performed using Power BI.
* Calculated columns such as Profit Margin and Loss Flag were created using DAX.
* Key measures like Total Revenue, Total Profit, and Profit Margin % were developed.
* Interactive dashboard built using KPI cards, bar charts, column charts, and line charts.
* Slicers were added for Region, Product, and Customer Type to enable dynamic filtering.
* Analysis identifies high-performing regions, profitable customer segments, and loss areas.
* Impact of discounts on profitability was evaluated using visual insights.
* The project demonstrates data analysis, DAX, and data visualization skills for business decision-making.
