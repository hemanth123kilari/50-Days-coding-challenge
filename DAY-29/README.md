Description

This project analyzes student marks using Python dictionaries. It calculates total and average marks, identifies the top scorer, and detects students who failed in any subject.

Concepts Covered
Dictionaries (nested dictionary handling)
Loops (for)
Conditional statements (if)
Built-in functions (sum(), max())
Lambda functions
Features
Stores student marks in structured format
Calculates total and average marks for each student
Identifies the top-performing student
Detects students failing in any subject (marks < 40)
Key Learnings
Working with nested dictionaries
Aggregating data using loops
Applying conditions on multiple values
Using lambda for efficient comparisons
Real-world data analysis using Python
