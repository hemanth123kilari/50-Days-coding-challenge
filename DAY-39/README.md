1. This project analyzes election data using Microsoft Excel.
2. The dataset includes constituency, party, candidate, votes, gender, and age.
3. Data cleaning was performed by removing duplicates and formatting columns.
4. Vote percentage was calculated using Votes / Total Votes.
5. Winners were identified based on highest votes in each constituency.
6. Pivot Tables were used to analyze party performance.
7. The party with highest total votes and most wins was identified.
8. Gender analysis was done to compare male and female vote share.
9. Age analysis included average, youngest, and oldest candidates.
10. A dashboard with charts and tables was created for visualization.
