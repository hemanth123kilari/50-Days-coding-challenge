The project consists of the following tables:

Users – Stores user details (username, email, join date)
Posts – Contains posts created by users
Likes – Tracks likes on posts
Comments – Stores user comments on posts
Friendships – Manages relationships between users

This project is a Social Media Database System built using SQL to simulate real-world social networking platforms. It manages user interactions such as posting content, liking posts, commenting, and maintaining friendships. The database is designed with proper normalization and relational integrity using primary and foreign keys.
