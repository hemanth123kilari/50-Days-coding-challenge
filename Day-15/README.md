Project Title

College Database Management using SQL

📖 Description

This project demonstrates the creation and management of a simple college database system using SQL. It includes tables for students, courses, and marks, along with various queries to analyze and retrieve data.

🗂️ Database Structure

The database consists of three main tables:

Students Table
Stores student details like ID, name, gender, city, and joining year
Courses Table
Contains course information such as course ID, name, and department
Marks Table
Records student marks and links students with courses using foreign keys
⚙️ Features Implemented
Database creation and table setup
Data insertion into tables
Use of Primary Keys and Foreign Keys
Data retrieval using:
SELECT statements
WHERE conditions
ORDER BY sorting
Aggregate functions:
COUNT()
AVG()
MAX()
MIN()
SUM()
🔍 Sample Queries
Filter students by city, gender, and join year
Retrieve marks above a certain threshold
Sort student and marks data
Calculate average, total, highest, and lowest marks
💡 Key Learnings
Understanding relational database design
Working with SQL queries and constraints
Applying filtering and sorting techniques
Using aggregate functions for data analysis
🚀 How to Run
Open MySQL / SQL environment
Copy and paste the SQL script
Execute step by step
Use SELECT queries to view results
📊 Output
Displays structured student, course, and marks data
Provides insights like average marks, total students, etc.
🛠️ Technologies Used
SQL (MySQL)
📌 Conclusion

This project helps in understanding how to design a relational database and perform basic to intermediate SQL operations for data analysis.
